Quick Start Guide

 

1.       Navigate to www.github.com. Sign-up if you are new or sign-in if you already have an account setup.

 

2.       In the upper-right corner of any page, use the + drop-down menu, and select New repository.

 

 

3.       Enter username.github.io as the repository name. Replace username with your GitHub username. For example, if your username is octocat, the repository name should be octocat.github.io.

 

4.     Under your repository name, click Settings.

  1. In the left sidebar, click Pages.
  2. Click Choose a theme.
  3. The Theme Chooser will open. Browse the available themes, then click Select theme to select a theme. It's easy to change your theme later, so if you're not sure, just choose one for now.
  4. After you select a theme, your repository's README.md file will open in the file editor. The README.md file is where you will write the content for your site. You can edit the file or keep the default content for now.
  5. When you are done editing the file, click Commit changes.
  6. Visit username.github.io to view your new website. Note: It can take up to 20 minutes for changes to your site to publish after you push the changes to GitHub.

ADDING A 2ND PAGE

1.       To add the 2nd page, an AboutMe page, type github.com/username. That will take you to the repository of the website.

2.       Download an HTML editor like Atom from Atom.io, and create the webpages with the Atom editor.

 

 

 

3.       Simply go to new file, click on Save as, and give a filename with html extension eg. Filename.html

4.       On the firstline of the new file, type html, hit Enter.

This will create the HTML boilerplate code, which can then be modified as needed.

 

 

The HTML code is added and modified between the body tags i.e. <body>.</body>.

To title the page, type About me between the title tags as shown in the image above.

 

5.       Name the file Index2.html and save changes and upload the saved file to github repository. You will click on Add file -> Upload files-> drag and drop your files and commit changes (save).

6.       Create a hyperlink on the main page (index.html) to link the added file(s).

 

On the main page between the body tags (<body> </body>), put the following anchor tag:

<a href=index2.html></a>.

 

7.       To add another page follow steps 1 through 6 again and change the name of the save file. i.e. index3.html, index4.html etc.

Also, change the words inbetween the title tags, to match the description of what you want the page to display.
